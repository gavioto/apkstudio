APK Studio is an Open-Source Cross-platform IDE for reverse-engineering Android applications, written using Qt framework. It is collaboratively developed by Vaibhav Pandey along with other contributor(s) and is currently being maintained and developed by Surendrajat and other contributor(s).

The main purpose of this project is to provide ROM makers with the ease of GUI for customization of pre-packaged applications and Android system. The source code for this application is hosted at Github.
